# Databricks Certified Associate Developer for Apache Spark - Study Guide Index

**Exam Format:** 45 multiple-choice questions | 90 minutes | 200 USD

**Total Score: 100%** — Each section below shows its weighted percentage on the exam.

---

## Study Guide Sections

### [Section 1: Apache Spark Architecture and Components](01_Architecture_and_Components.md) — **20%**
- Objective 1: Identify the advantages and challenges of implementing Spark
- Objective 2: Identify the role of core components of Apache Spark's Architecture
- Objective 3: Describe the architecture of Apache Spark (DataFrame, Dataset, SparkSession, caching, storage levels, garbage collection)
- Objective 4: Explain the Apache Spark Architecture execution hierarchy
- Objective 5: Configure Spark partitioning in distributed data processing (shuffles and partitions)
- Objective 6: Describe the execution patterns of Apache Spark (actions, transformations, lazy evaluation)
- Objective 7: Identify the features of Apache Spark Modules (Core, Spark SQL, DataFrames, Pandas API on Spark, Structured Streaming, MLlib)

### [Section 2: Using Spark SQL](02_Spark_SQL.md) — **20%**
- Objective 1: Utilize common data sources (JDBC, files, etc.) to efficiently read from and write to Spark DataFrames
- Objective 2: Execute SQL queries directly on files (ORC, JSON, CSV, Text, Delta)
- Objective 3: Save data to persistent tables with sorting and partitioning
- Objective 4: Register DataFrames as temporary views in Spark SQL

### [Section 3: Developing Apache Spark DataFrame/DataSet API Applications](03_DataFrame_API.md) — **30%**
- Objective 1: Manipulate columns, rows, and table structures
- Objective 2: Perform data deduplication and validation operations
- Objective 3: Perform aggregate operations (count, approximate count distinct, mean, summary)
- Objective 4: Manipulate and utilize Date data type
- Objective 5: Combine DataFrames (joins and unions)
- Objective 6: Manage input and output operations (reading, writing, schemas)
- Objective 7: Perform operations on DataFrames (sorting, iterating, printing schema, conversion)
- Objective 8: Create and invoke user-defined functions (UDFs)
- Objective 9: Describe different types of variables (broadcast variables and accumulators)
- Objective 10: Describe the purpose and implementation of broadcast joins

### [Section 4: Troubleshooting and Tuning Apache Spark DataFrame API Applications](04_Troubleshooting_and_Tuning.md) — **10%**
- Objective 1: Implement performance tuning strategies & optimize cluster utilization
- Objective 2: Describe Adaptive Query Execution (AQE) and its benefits
- Objective 3: Perform logging and monitoring of Spark applications

### [Section 5: Structured Streaming](05_Structured_Streaming.md) — **10%**
- Objective 1: Explain the Structured Streaming engine
- Objective 2: Create and write Streaming DataFrames with output modes and sinks
- Objective 3: Perform basic operations on Streaming DataFrames
- Objective 4: Perform Streaming Deduplication with and without watermark

### [Section 6: Using Spark Connect to Deploy Applications](06_Spark_Connect.md) — **5%**
- Objective 1: Describe the features of Spark Connect
- Objective 2: Describe different deployment mode types (Client, Cluster, Local)

### [Section 7: Using Pandas API on Apache Spark](07_Pandas_API.md) — **5%**
- Objective 1: Explain the advantages of using Pandas API on Spark
- Objective 2: Create and invoke Pandas UDFs
